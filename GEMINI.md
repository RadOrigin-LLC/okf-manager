# GEMINI.md — okfm (OKF Manager)

Operating instructions for this repo are in **`AGENTS.md`** (canonical, cross-harness). Read it first.

In short: the engine is the `okf` CLI — invoke it as `okf <command>`, `python okf.pyz <command>` (build with `python build-okf-pyz.py`), or `python <repo>/scripts <command>`. The operating manual is `USAGE.md`; task guidance is in `skills/<name>/SKILL.md`. `enrich` needs Gemini's own web-fetch tool; the engine never goes online.
